--- Deskripsi ---
Project ini merupakan sebuah website portal Penerimaan Peserta Didik Baru (PPDB) yang dibuat menggunakan HTML, CSS, dan JavaScript untuk memenuhi tugas UTS mata kuliah Desain & Pemrograman Web. Website ini menggunakan konsep multi-page modern, sehingga terdiri dari tiga halaman utama yaitu Beranda, Alur Pendaftaran, dan Form Pendaftaran. Pada halaman Beranda (index.html), terdapat tampilan sambutan untuk calon siswa baru serta informasi mengenai jadwal pelaksanaan PPDB yang disusun dalam bentuk tabel. Tabel tersebut dibuat menggunakan elemen HTML seperti <table>, <thead>, dan <tbody> agar data jadwal terlihat lebih rapi, jelas, dan mudah dipahami pengguna. Untuk halaman Alur Pendaftaran (alur.html), website menampilkan gambar ilustrasi beserta tahapan proses pendaftaran yang dibuat menggunakan ordered list (<ol>). Bagian ini dibuat lebih menarik dengan tampilan step card modern, sehingga setiap langkah terlihat lebih jelas, rapi, dan nyaman dilihat. Pada halaman Form Pendaftaran (form.html), tersedia form untuk mengisi nama lengkap, NIK, dan jalur pendaftaran. Form ini dilengkapi dengan validasi JavaScript supaya pengguna tidak bisa mengirim data jika masih ada kolom yang kosong. Selain itu, kolom NIK harus berjumlah tepat 16 digit. Jika ada kesalahan, sistem akan menampilkan toast notification sebagai pesan error, sedangkan jika data sudah benar maka akan muncul notifikasi berhasil dan form akan otomatis ter-reset. Dari sisi tampilan, website ini menggunakan CSS eksternal dengan desain soft gradient modern, ditambah efek hover pada menu navigasi dan tombol, glow effect pada input saat difokuskan, serta warna tabel selang-seling (zebra striping) agar lebih enak dibaca. Struktur folder project juga sudah dibuat rapi sesuai modul, yaitu memisahkan file HTML, CSS, JavaScript, dan gambar ke dalam folder asset. Secara keseluruhan, project ini dibuat agar proses pendaftaran siswa baru dapat dilakukan secara online dengan tampilan yang modern, rapi, dan mudah digunakan, sehingga memberikan pengalaman pengguna yang lebih nyaman.
